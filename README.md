## How to use:

fork the repo and clone it to ur local machine and run:


```bash
yarn
or
npm i

```

This will install all the dependecies, then you could run:

```bash
yarn run dev
or
npm run dev

```

This will spin up a dev server on ur **localhost**

Enjoy ... ❤️
